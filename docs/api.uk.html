Довідник API

Ендпоінти
GET / — веб-інтерфейс
GET /api/v1/system-state — поточний стан
POST /api/v1/apply-mechanism — запуск агента, збереження стану та історії
GET /api/v1/agent-runs?limit=20&offset=0 — історія запусків
Моделі (Pydantic)

MechanismInput

{ "target_goal": "string (min 3)" }

MechanismResponse

{
  "newState": { /* SystemState */ },
  "explanation": "string",
  "explanation_details": { "ResourceType": number }
}

SystemState

{
  "components": [{ "id": "string", "name": "string", "status": "string" }],
  "resources": [{ "id": "string", "name": "string", "type": "ResourceType", "value": 0.0 }]
}
